Java Programming in Alice

Junior Year Quarter 1

Alice is a Programming Integrated Development Environment (IDE) designed to teach the Java programming language and let students quickly create 3D models and environments that are programmable and playable. They can be scripted movies or interactive games that listen to keystrokes and mouse movements.

Topics students learn this quarter are:

  1. Storyboarding and Writing algorithms
  2. Writing methods
  3. Learning Datatypes and expressions
  4. Learning Control Flow
  5. Arrays
  6. Event handling and listeners
